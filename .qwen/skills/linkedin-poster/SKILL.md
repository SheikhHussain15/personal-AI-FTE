# LinkedIn Poster Skill

Automatically post to LinkedIn for business promotion.

## Prerequisites

### 1. Install Playwright

```bash
pip install playwright
playwright install chromium
```

### 2. Setup Session Directory

```bash
mkdir -p AI_Employee_Vault/Scripts/linkedin_session
```

### 3. Authenticate

1. Run poster script first time
2. Login to LinkedIn when browser opens
3. Session saved for future use

## Quick Start

### Create Post

```bash
python .qwen/skills/linkedin-poster/scripts/linkedin_poster.py \
  --vault ../AI_Employee_Vault \
  --action post \
  --content "Excited to announce our new service..."
```

### Schedule Post

```bash
python .qwen/skills/linkedin-poster/scripts/linkedin_scheduler.py \
  --vault ../AI_Employee_Vault \
  --time "2026-02-27T09:00:00" \
  --content "Morning motivation post..."
```

## Configuration

### Environment Variables

```bash
LINKEDIN_SESSION_PATH=/path/to/linkedin_session
LINKEDIN_POST_APPROVAL=true  # Require approval before posting
```

### Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `--approval` | true | Require human approval |
| `--schedule` | none | Schedule time for post |
| `--hashtags` | auto | Auto-generate hashtags |

## How It Works

```
1. Content created (by AI or human)
2. If approval required: create Pending_Approval file
3. Human approves (moves to Approved/)
4. Playwright logs into LinkedIn
5. Creates post with content + hashtags
6. Screenshot saved to Logs/
7. Result logged
```


## Approval Workflow

Posts require approval before publishing:

1. AI creates post draft in `Pending_Approval/`
2. Human reviews content
3. Moves to `Approved/` to publish
4. Moves to `Rejected/` to discard

## Best Practices

- Post 2-3 times per week
- Mix content types (updates, tips, news)
- Use relevant hashtags (3-5)
- Include call-to-action
- Respond to comments within 24 hours

## Security Notes

- LinkedIn session is sensitive - keep secure
- Never commit session files to git
- Use approval workflow for all posts
- Respect LinkedIn's Terms of Service

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Login failed | Re-authenticate, clear session |
| Post failed | Check LinkedIn is accessible |
| Content truncated | Keep posts under 3000 characters |
