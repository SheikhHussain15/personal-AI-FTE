## How to Use (Quick Reference)

### Start the System

```batch
# Terminal 1 - Filesystem Watcher
cd AI_Employee_Vault\Scripts
python filesystem_watcher.py --vault ..

# Terminal 2 - Orchestrator
cd AI_Employee_Vault\Scripts
python orchestrator.py --vault .. --watch
```

### Process Items with Qwen Code

1. Drop file in `AI_Employee_Vault\Inbox\`
2. Wait for watcher to process (30 seconds)
3. Open Qwen Code IDE
4. Prompt: "Process items in Needs_Action folder"

```batch
cd bronze-tier
```
